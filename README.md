# Next.js Resume

A single page resume built with Next.js and Tailwind CSS

## How to Run 

npm install
npm run dev

## Features Included

- [x] Resume built in NExt.js App router using JSX
- [x] Styled with Tailmind CSS utility classes
- [x] Qualification Summary
- [x] Education section
- [x] Work Experience
- [x] Coursework projects
- [x] Skills and certifications
- [x] Professional organizations
- [x] Semetic HTML elements used throughout
- [x] No console errors

## Known ISsues / Limitations

- No mobile layout
