# Next.js Resume

A one page resume for Mackenzie Prete, built with Next.js and Tailwind CSS. Showcases education, work experience, coursework projects, and technical skills.

## Setup

1. Install Node.js if you don't have it
2. Download this project and open it in VS Code

## How to Run

1. Open the terminal in VS Code
2. Type `npm install` and hit enter
3. Type `npm run dev` and hit enter
4. Go to http://localhost:3000 in your browser

## How to Build

1. Type `npm run build` in the terminal
2. Type `npm start` to run it
   
## Features Included

- [x] Resume built in Next.js App Router using JSX
- [x] Styled with Tailwind CSS utility classes
- [x] Qualification Summary
- [x] Education section
- [x] Work Experience
- [x] Coursework Projects
- [x] Skills and Certifications
- [x] Professional Organizations
- [x] Semantic HTML elements used throughout
- [x] No console errors
